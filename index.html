Welcome to Etch-A-Sketch!

Express your creativity, this is cross over between etch-a-sketch toy and drawing pad. Pick up a pen and scribble on the grid.

To reset grid select size of the grid and reset button. Grid sizes:

  • Tiny 16 x 16 grid
  • Small 32 x 32 grid
  • Medium 64 x 64 grid
  • Large 100 x 100 grid
